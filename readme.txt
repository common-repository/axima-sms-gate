== Description ==

Reliable and cheap SMS gate, which sends guaranteed SMS to whole world. Support for delivery reports included.

== Installation ==

= Requires: =
Wordpress 4.5 or higher, PHP at least 5.4, WooCommerce plugin installed.

= Installation steps =
1. Install plugin in Wordpress and activate it
2. Login to your sms.sluzba.cz account through plugin settings
3. Change SMS texts in plugin settings

== Screenshots ==

1. Dashboard, which provides some statistics, information about your account and possibility to send custom sms.
2. Logs, where you can see list of all sent messages and check if they were delivered. Check is automatically done every hour, but you can check it also manually.
3. Settings page, where you enable or disable sms messages for different order statuses. You can also use placeholders for more customized message.
